GolHub利用規約

株式会社Golf Hub（以下「当社」といいます。）は、この「GolfHub利用規約」（以下「本規約」といいます。）を定め、これにより「GolHub」（名称が変更された場合は変更後のものとし、以下「本サービス」といいます。）を提供します。

第1条 （規約の適用）
1.本規約は、本サービスの利用に関する当社との間の一切の関係に適用されます。　　　本規約の内容に同意しない場合、本サービスを利用することはできません。
2. 本規約の内容と、本サービス情報サイト（第2条第4号に定めるとおりとします。）その他の本規約外における本サービスの説明等とが異なる場合、本規約の規定が優先して適用されるものとします。

第2条 （用語の定義）
本規約において、次の各号に掲げる用語の意味は、当該各号に定めるとおりとします。
1.  利用契約：　当社から本サービスの提供を受けるための本規約に基づく契約をいいます。
2. 利用者：　当社との間で利用契約を締結した者をいいます。
3.  本サービスサーバ：　当社が本サービスを提供するために設置する電子計算機(サーバ)をいいます。
4.  本サービス情報サイト：　本サービスに関する情報を掲載した当社のインターネットウェブサイト＜ https://golhub.co、https://golf-hub.co＞（当該URL配下のインターネットウェブサイトを含み、当社がこのURLを変更した場合は、変更後のURLとします。）をいいます。
5.  本サービスアプリ：　本サービスを利用するために必要となる本サービス専用のアプリケーションソフトウェアをいいます。
6. 対応端末：　本サービスを利用できるものとして、当社が別途本サービス情報サイト上で指定するバージョンのiOSを搭載する端末をいいます。
7. アップロード：　当社が別途指定するアングル等より撮影し対応端末に保存した動画のデータファイル（以下「動画ファイル」といいます。）を本サービスアプリを介して本サービスサーバに送信することをいいます。
8. 本サービスコンテンツ：　本サービスを通じて利用者に提供される、次号に定める利用者コンテンツの診断結果、文章、画像、動画、音楽その他の情報及びそれらの組み合わせをいいます。
9. 利用者コンテンツ：　次号に定める診断の対象となる、利用者がアップロードした動画ファイルをいいます。
10. 診断：　利用者コンテンツからゴルフスイングの軌道、スイング時の姿勢等を解析し、AI（人工知能）を用いて当該スイングの軌道、姿勢等に対する評価をすることをいいます。
11. コーチ：　本サービスコンテンツの一部を制作した者をいいます。
12. コーチ制作コンテンツ：　本サービスコンテンツのうち、コーチが制作したコンテンツをいいます。
